# Online Food Ordering System

Professional MERN stack project setup for an online food ordering system.

## Stack

- React + Vite
- Tailwind CSS
- React Router
- Axios
- Firebase Authentication
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT

## Setup

1. Install dependencies:

```bash
npm install
```

2. Create `.env` from `.env.example` and add Firebase, MongoDB, and JWT values.

3. Run the frontend:

```bash
npm run dev
```

4. Run the backend:

```bash
npm run dev:server
```

## Current Scope

This milestone includes only project setup:

- Navbar and footer
- Home, login, register, protected profile, and 404 pages
- Loading spinner
- Firebase email/password auth
- Firebase Google login
- Logout
- Protected routes
- Axios API service
- Express server
- MongoDB connection helper
- JWT token creation
- JWT verification middleware

Food ordering features are intentionally not implemented yet.
